# Interactive Sales Performance Dashboard

## Project Overview
This project is an interactive sales dashboard built in Microsoft Excel to analyse sales performance, revenue trends, and business KPIs. The dashboard was designed to transform raw sales data into meaningful business insights using Pivot Tables, Pivot Charts, slicers, and Excel formulas.

The project focused not only on dashboard creation, but also on realistic data cleaning and preparation processes commonly required in business reporting environments.

# Data Cleaning & Preparation Process

Before building the dashboard, the dataset required extensive cleaning and standardisation to ensure accurate reporting and analysis.

### Data Cleaning Steps Performed

#### 1. Standardised Date Formats
The dataset contained inconsistent date formats such as:
- `2024-01-15`
- `15/01/2024`
- `Jan 15 2024`

These were converted into a consistent Excel date format to enable accurate monthly and quarterly trend analysis.

---

#### 2. Corrected Inconsistent Category Names
Several fields contained inconsistent naming conventions such as:
- “North”
- “north”
- “NORTH”

These values were standardised to ensure categories grouped correctly within Pivot Tables and dashboard visuals.

---

#### 3. Cleaned Extra Spaces & Formatting Issues
Used Excel functions such as:
- `TRIM()`
- `CLEAN()`
- Find & Replace

to remove:
- extra spaces
- hidden characters
- inconsistent text formatting

---

#### 4. Converted Dataset into an Excel Table
The raw dataset was converted into a structured Excel Table to:
- improve scalability
- enable dynamic Pivot Table updates
- simplify dashboard maintenance

---

#### 5. Checked for Duplicate Records
Reviewed the dataset for duplicate transactions and removed unnecessary duplicate entries to maintain data integrity.

---

#### 6. Created Calculated Fields
Additional calculated fields were added to support reporting and KPI analysis, including:
- Month Column
- Quarter Column
